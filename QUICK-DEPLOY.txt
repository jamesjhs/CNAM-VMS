## ⚡ Copy-Paste Commands (Run in Order)

```bash
# 1. SSH to server
ssh root@cnamvms.jahosi.co.uk

# 2. Navigate to app directory
cd /var/node/cnamvms.jahosi.co.uk-3001

# 3. Backup database
cp data/cnam.db data/cnam.db.backup.$(date +%Y-%m-%d)

# 4. Pull code
git fetch origin && git checkout security-update && git pull origin security-update

# 5. Install deps
npm ci

# 6. Build
npm run build

# 7. Restart
pm2 restart cnam-vms --update-env

# 8. Verify
pm2 logs cnam-vms --lines 50
```
